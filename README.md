# big-data-flight-delay-analysis-pyspark

## Description
This project analyzes flight delay data using PySpark to demonstrate big data processing and scalability.

## Dataset
The dataset is taken from Kaggle:
https://www.kaggle.com/datasets/hrishitpatil/flight-data-2024

## Technologies Used
- Python
- PySpark
- Google Colab
- Matplotlib

## Key Features
- Data cleaning
- Delay analysis
- Airline performance insights
- Scalable processing using PySpark
